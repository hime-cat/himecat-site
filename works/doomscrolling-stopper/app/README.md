# 反芻ストッパー / Doomscrolling Stopper

頭から離れないことを、いったん書き出して整理するための小さなWebアプリです。

AI APIには接続せず、入力内容からChatGPTなどへ渡すための文章を生成します。AIの返答を貼り付けると、ブラウザ内に整理として保存できます。

## 使い方

1. 頭の中で回っていることを箇条書きで書く
2. 生成された文章をコピーする
3. ChatGPTなどに貼る
4. AIの返答を戻して保存する
5. 必要なときだけ「見返す」から確認する

## 保存について

現在はローカル保存のみです。データは使っているブラウザに保存されます。

保存される整理は最新15件までです。15件を超えた場合、古い整理から自動で削除されます。

SupabaseなどのDB連携は、将来的に「別の端末でも見られるようにする」ための任意機能として検討します。

## 公開

静的HTML/CSS/JavaScriptのみで動作します。
現在の公開先は himecat ホームページ配下の以下です。

```text
https://himecat.com/works/doomscrolling-stopper/app/
```

PWA対応済みです。対応ブラウザではホーム画面への追加や、キャッシュ済み画面のオフライン表示ができます。

画像素材は `images/` にまとめています。`stopper-signal.svg` はアイコンとブランドマーク、`ogp.png` はOGP画像として使います。
